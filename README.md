# Firebase Authentication iOS 🚀

A modern iOS authentication app built using **SwiftUI** and **Firebase Authentication**.

---

## ✨ Features

- 📧 Create User with Email & Password
- 🔐 Sign In with Email & Password
- 🔑 Forgot Password Flow
- 🌍 Google Sign-In Integration
- 🔥 Firebase Authentication
- 🎨 Clean SwiftUI UI
- ⚡ Async/Await Support

---

## 🛠 Tech Stack

- SwiftUI
- Firebase Authentication
- Google Sign-In SDK
- MVVM Architecture
- Async/Await

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/26081yogesh/Firebase_Authentication_IOS.git
```

### 2. Open the project

```bash
cd Firebase_Authentication_IOS
open TestingFirebaseIOS.xcodeproj
```

### 3. Add Firebase Configuration

Add your own:

```text
GoogleService-Info.plist
```

inside the Xcode project.

---

## 🔥 Firebase Setup

1. Create a Firebase project
2. Enable Authentication
3. Enable:
   - Email/Password
   - Google Sign-In
4. Download `GoogleService-Info.plist`
5. Add it to the project
